hexo主题next及其配置

发表于 2021-10-30 更新于 2021-10-31 分类于 hexo ， hexo-theme-next 阅读次数： 本文字数： 2k 阅读时长 ≈ 2 分钟

hexo 主题

hexo 主题
next 文档

更换主题为next

  1. 下载next主题
    在主目录下，以下方式任选一种
    npm：npm install hexo-theme-next
    git:git clone https://github.com/next-theme/hexo-theme-next themes/next

  2. 修改_config.yml文件
    将theme的内容改为next
    theme: next

个性化配置

以下内容是修改主题配置文件（注意和站点配置文件区分）
npm的方式：node_modules/hexo-theme-landscape/_config.yml文件
git的方式：themes/next/_config.yml文件

scheme 方案

next主题可以选择4种scheme，这里选择的是Gemini

menu 导航菜单

menu下面的选项，去掉前面的注释即可开启菜单

avatar 自定义头像

  1. 将头像放在hexo站点的目录source/uploads/下并修改相应的url/uploads/xxx.jpg

  2. 将头像放在next主题的目录/source/images下并修改相应的url/images/xxx.jpg

  3. 图片的绝对地址

social 社交账号

social下面的选项，去掉前面的注释并修改相应的url即可

footer 页脚

footer下面可以自定义图标，copyright，服务支持，备案信息等等，这里不做展示

reward 打赏设置

reward_settings下的enable先设置为true
reward下面可以设置微信收款码，支付宝收款码等等
将收款码图片复制到themes/next/source/images目录下，修改相应的url的内容即可
