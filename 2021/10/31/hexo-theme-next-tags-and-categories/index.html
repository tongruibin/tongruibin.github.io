标签和分类

发表于 2021-10-31 分类于 hexo ， hexo-theme-next 阅读次数： 本文字数： 474 阅读时长 ≈ 1 分钟

添加「标签」页面

  1. 在Hexo 站点目录下。使用 hexo new page 新建一个 tags 页面
    hexo new page tags

  2. 编辑source/tags/index.md文件
    在文件的Front-matter的title的date下面增加内容type: tags
    如果有集成评论服务，页面也会带有评论。可以增加内容comments: false关闭

  3. 修改主题配置文件的 menu，开启tags功能

以后在文章的Front-matter部分增加tags就会自动打上标签

添加「分类」页面

  1. 在Hexo 站点目录下。使用 hexo new page 新建一个 categories 页面
    hexo new page categories

  2. 编辑source/categories/index.md文件
    在文件的Front-matter的title的date下面增加内容type: categories
    如果有集成评论服务，页面也会带有评论。可以增加内容comments: false关闭

  3. 修改主题配置文件的 menu，开启categories功能

以后在文章的Front-matter部分增加categories就会自动分类
