next主题进阶应用

发表于 2021-10-31 更新于 2021-11-01 分类于 hexo ， hexo-theme-next 本文字数： 1.5k 阅读时长 ≈ 1 分钟

添加「标签」页面

  1. 在Hexo 站点目录下。使用 hexo new page 新建一个 tags 页面
    hexo new page tags

  2. 编辑source/tags/index.md文件
    在文件的Front-matter的title的date下面增加内容type: tags
    如果有集成评论服务，页面也会带有评论。可以增加内容comments: false关闭

  3. 修改主题配置文件的 menu，开启tags功能

以后在文章的Front-matter部分增加tags就会自动打上标签

更换主题为next

  1. 下载next主题
    在主目录下，以下方式任选一种
    npm：npm install hexo-theme-next
    git:git clone https://github.com/next-theme/hexo-theme-next themes/next

  2. 修改_config.yml文件
    将theme的内容改为next
    theme: next
